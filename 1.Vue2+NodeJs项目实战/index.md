
# Vue2.0+NodeJS项目实战

## 1 课程介绍

### 1.1导学

* Vue框架对比
  1. [Vue](https://cn.vuejs.org)和[React](https://reactjs.org)目前都使用了Virtual DOM
  2. Vue
    * [github](https://github.com/vuejs/vue)
    * 模板和渲染函数的弹性选择
    * 简单的语法及项目创建
    * 更快的渲染速度和更小的体积
  3. React
    * [github](https://github.com/facebook/react)
    * 更适用于大型应用和更好的可测试性
    * 同时适用于Web端和原生App
    * 更大的生态圈带来的更多支持和工具
  4. Vue和React相同点
   * 利用虚拟DOM实现快速渲染
   * 轻量级
   * 响应式组件
   * 服务器端渲染
   * 易于集成路由工具，打包工具以及状态管理工具
   * 优秀的支持和社区
